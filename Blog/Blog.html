Exam

Q1 : What is JavaScript and how is it used in web development?

Answer : JavaScript is a versatile programming language primarily used for creating dynamic and interactive content on websites. It allows developers to manipulate web pages in real-time, handle events, and perform asynchronous operations, enhancing the user experience.

Q2 : What are closures in JavaScript?

Answer : Closures are functions that retain access to their lexical scope even when the function is executed outside that scope. They are useful for creating private variables and functions, ensuring data encapsulation.

Q3 : What is the difference between let, const, and var in JavaScript?

Answer : `var` is function-scoped, `let` is block-scoped, and `const` is block-scoped with an immutable binding. `let` and `const` provide better scoping rules and help prevent common bugs associated with `var`.

Q4 : What is the event loop in JavaScript?

Answer : The event loop is a mechanism that allows JavaScript to perform non-blocking operations by offloading tasks to the browser, then running the tasks in the queue once the main stack is empty. This ensures efficient handling of asynchronous operations.

Q5 : How does prototypal inheritance work in JavaScript?

Answer : Prototypal inheritance allows objects to inherit properties and methods from other objects. Each object has a prototype, which is a reference to another object, enabling shared properties and methods across instances.
